Angular Routing

It maps specific URL paths to corresponding Angular components, enabling dynamic loading and display of content without reloading the entire page. This system handles everything from basic navigation, route parameters, and redirects to more advanced features like lazy loading and route guards.

Router Module

Angular's routing functionality is provided by the RouterModule from the angular/router package. To set up routing, its needed to import RouterModule in the main module.

The most basic route in Angular need:

  • Path
  • Componets that wll be mached with that specific route

Navigating into routes or componets

Exist many ways to navigate into routes components the most common ways are with a function that navigates to an specified route and the second one is with routerLink directly in the HTML.

https://angular-training.com/helloAngular

<button
  class="btn btn-primary"
  routerLink="/">
  Go Home
</button>
OR
<button
class="btn btn-primary"
(click)="navigateToHome()">
Go Home
</button>
TS
constructor(private router : Router)
navigateToHome() {
  this.router.navigate(['/')
}
